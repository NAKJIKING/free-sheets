## 한 줄 결론

**"새로 조판된 CC0/자유 라이선스 악보를 우선 수집하고, 옛 악보 스캔본(IMSLP)은
판본 층위 권리를 확인한 뒤"** — 이 순서가 법적으로 가장 안전하다.

## 2. 법적 기준 핵심 (검증 완료)

- **작곡가 저작권**: 한국·EU 사후 70년. 미국은 **1930년 이전 최초 출판이면 무조건 PD**
  (컷오프 매년 1년씩 전진 — 2026년 현재 1931년 이전). 미국 외 출판물은 URAA 복원
  때문에 "표시 누락·미갱신 → PD" 경로가 안전하지 않음.
- **판면권(조판 저작권)** — 작곡가 저작권과 **별개의 권리**:
  - 영국: 출판 후 25년 (CDPA s.15) / 독일: 학술판 25년 (UrhG §70) / EU 지침: 최대 30년
  - 미국·캐나다: 조판만으로는 저작권 불성립 (스캔·재인쇄에 신규 권리 없음)
  - → **현대 출판사 판본(비평판·원전판)의 스캔은 EU·영국에서 재배포 위험.**
    IMSLP 자체도 "출판 25년 경과 또는 독일 PD" 규칙을 적용 중
- **안전한 IMSLP 선별 기준**: 원판 자체가 오래된 스캔(1930년 이전 출판 원판)만,
  현대 비평판·원전판 스캔은 제외
- **편곡·현대판**: 편곡자의 별도 저작권 발생 — 작곡가·작사가·**편곡자** 전원
  사후 70년 확인 필요

## 3. 곡별 검증 체크리스트

1. 작곡가·작사가·편곡자 전원 사망연도 + 70년 경과?
2. 미국 기준: 최초 출판 1931년 이전?
3. 스캔본이면: 원판 출판연도·출판사·판본 성격 (원판 재인쇄 ✓ / 현대 비평판 ✗)
4. 라이선스 태그와 파일 내부 저작권 표기 대조 (PDMX 사례: 12.29% 불일치)
5. 카탈로그에 기록: 출처 URL · 라이선스 · 판본 정보 · 고지 문구

## 4. 실행 계획 (0원 예산, GitHub 기반)

- **1단계 (완료)**: Mutopia 수집기 가동 — 악기별 자동 수집 + catalog.json
- **2단계 (다음)**: OpenScore Lieder GitHub 일괄 클론 → CC0라 무조건 안전.
  단 MuseScore 형식이라 PDF 변환 단계 필요 (Actions에서 MuseScore CLI 렌더링)
- **3단계**: PDMX no_license_conflict 서브셋에서 악기별 인기곡 선별
- **4단계**: CPDL 합창곡 (CPDL 라이선스 곡만 필터)
- **5단계**: IMSLP는 §2 선별 기준 통과분만 수동 큐레이션
- **앱 고지 문구**: 곡 상세에 "출처·라이선스" 표시 + 설정에
  "일부 악보는 국가에 따라 저작권 상태가 다를 수 있습니다" 고지
